## Featured Projects

### Prismarine (5th Place)

**5th Place @ CTRL+HACK+DEL · Feb 2026**

Interactive research exploration platform that turns paper collections into a 3D similarity graph. Built with `react-force-graph-3d` and `Three.js`, backed by a `FastAPI` service aggregating arXiv, PMC, and Semantic Scholar with resilient fallbacks. Includes embedding-based linking (`cosine similarity + kNN`), ElevenLabs narration, and Docker Compose deployment via Coolify.

[Event / Devpost](https://devpost.com/software/prismarine) · [Live Demo](https://prismarine.yvesdonato.com/) · [GitHub](https://github.com/YvesDonato/YorkHackathon2026)

**Tech:** React, Three.js, react-force-graph-3d, FastAPI, Semantic Scholar, ElevenLabs, Docker Compose, Coolify

### NeuRizz (2x Category Winner)

**2x Category Winner @ Hackville · Jan 2026**

Real-time social simulation platform for neurodivergent users built with SvelteKit and Three.js, focused on a non-stimulating and accessible 3D interface. Integrated Google AI and ElevenLabs for low-latency voice agents, live sentiment analysis, conversational scoring, and emotive text-to-speech. Deployed on a secure self-hosted stack with Coolify and MongoDB.
